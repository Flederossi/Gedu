# Gedu
A simple application to create Pixel-Art by writing code.

## Commands
```
fill > [color]: Set pixel color
size > [size]: Set size of canvas
pix > [x],[y]: Draw pixel
rect > [x1],[y1],[x2],[y2]: Draw rectangle
```
